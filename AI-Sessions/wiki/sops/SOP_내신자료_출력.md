# SOP: 내신 자료 출력

담당: [[2_교무부]] (출력 자원은 [[4_행정운영부]])

## 절차

1. 모의고사/내신 자료 폴더에서 해당 학교·학년 자료 탐색 (파일명 패턴 준수)
2. 학교-출판사 매핑표로 교과서 확인 (학년별 변동 주의)
3. 출력 규칙:
   - **강의용교안 → 컬러** (C3330)
   - **리뷰테스트 문제지 → 흑백** (iR-ADV 6255, 기본)
   - 둘은 항상 **동반 출력**
4. 공유폴더(\\desktop-07gd04s\일곡점 공유폴더2)에 정리

관련: [[학원 자원]] · [[참고자료 색인]]
