# 2. 교무부

- **담당자**: 선영 부장
- **미션**: 커리큘럼·진도·시험 대비 자료를 학생 맞춤으로 운영

## 기능 구조 (2026-06-07 설계)

교무부 = 5기능 1루프:
1. **컨텐츠 개발 (3트랙)** — 정규·내신·수능. *"무엇을"은 교무부, 엔진(도구)은 [[5_시스템개발부]]* (R1)
2. **학생별 배정·설계** — 진도·숙제 분량·자료 배정 → **[[교무부_운영룰]]** (#1·#2·#6, 조정 기준표는 제안 단계)
3. **숙제 검사·피드백** — 자동(단어앱·빠른채점)/수동(조교 채점) → **[[교무부_운영룰]]** #3
4. **분석·전략** — 학교 시험지 분석 → 성적향상 로드맵 → **[[내신_성적향상_로드맵_방법]]** ⭐ 핵심 IP. **✅구축: [[내신_분석_시스템]]**(시험 241개 분석·학교 36개 프로파일·출처 매칭 코퍼스 27,000)
5. **예상문제 출제 (족집게 대비)** — 3개년 기출 분석 → 패턴 유사 예상문제 대량 생성·드릴 (④와 ①을 잇는 다리) → **[[내신_성적향상_로드맵_방법]]**. (토대 구축됨: 매칭으로 출처·변형 식별 → [[내신_분석_시스템]])

DB는 콘텐츠=교무부 / 기술(병합·푸시)=[[5_시스템개발부]] (R2).

## 담당 업무

- 학생별 커리큘럼/진도 관리 (학생_마스터.xlsx 기준)
- 내신 대비 자료 준비·출력 → [[SOP_내신자료_출력]]
- 모의고사 숙제 설계 → [[SOP_모의고사_숙제]]
- 교과서 단원별 문법 매핑 관리 (출판사별 1~8단원)
- 지문 코퍼스 관리(모의고사·모아닷·부교재 약 27,000개) → [[내신_분석_시스템]]
- 시험지 분석·출처 매칭·출제 프로파일 운영 → [[내신_분석_시스템]] (데이터=`C:\시험지분석`)

## 핵심 규칙

- 학교-출판사 매핑은 학년별로 다를 수 있음 — 매핑표 항상 확인
- 강의용교안=컬러, 리뷰테스트 문제지=흑백 동반 출력
- 모의 숙제 표준: 18지문(20·21·22·23·24·29~40·41~42), 매일 3지문 × 6일 + 리뷰테스트·단어 인증
