# 4. 행정운영부

**미션**: 원생 명단·출결·물리적 자원 등 학원의 기반 운영

## 담당 업무

- CRM 원생 명단 관리 — 기본 필터는 본인 담당만 보이므로 **담당T(전체)** 로 조회. (원생 수는 CRM 실시간 — 위키에 하드코딩 안 함)
- 출결·시간표 관리 (학생_마스터.xlsx)
- 출력·인쇄 자원 운영 → [[학원 자원]]
- 파일·폴더 정리 (공유폴더 \\desktop-07gd04s\일곡점 공유폴더2)

## 핵심 규칙

- 흑백 기본: iR-ADV 6255 / 컬러: C3330
- 학생 데이터 변경(추가·퇴원)은 마스터 파일과 CRM 양쪽 동기화

## 등록 처리 흐름 (깔때기의 '등록' → OT)

1. 학생 정보 받아 **CRM 기입**
2. **등원 필독사항** 발송(학부모·아이)
3. **홈페이지 회원가입**
4. **홈페이지에서 학부모 직접 결제**(CRM 결제 기능)
5. 결제 완료 → **OT** 진행

(← [[3_상담소통부]] 레테·상담 확정 / 유입 ← [[6_브랜드홍보부]])

## 결제 · 출결 · 시간표

- **결제**: CRM 결제 기능 + 등록 시 홈페이지에서 학부모 직접 결제. (미납 추적·알림 방식 = 원장 확인)
- **출결**: 등·하원 시 학생이 **아이패드에 본인 번호 입력** → CRM 자동 기록(등·하원 시각 = 학습 시간). *현 CRM 종속 — 새 사업 이식 시 자체 개발 필요 ([[0_본사_전략실]] 이식성 과제).*
- **시간표**: 개인별 주 1회 → **원장이 학생과 직접 협의**해 수동 편성·변경.
