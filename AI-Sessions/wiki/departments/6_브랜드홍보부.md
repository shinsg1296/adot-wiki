# 6. 브랜드홍보부 (신설)

**미션**: HAWE 브랜드로 학원을 알리고 **신규 문의를 만든다** (학원의 '입구').

기존 1~5부서는 전부 *기존 학생을 돌보는* 일이라 신규 유입 담당이 비어 있어 신설. (근거: [[의사결정_조직구조]])

## 유입 경로 & 전략 (2026-06-07)

| 순위 | 경로 | 전략 |
|---|---|---|
| 1 | **친구 소개(입소문)** — 가장 큼 | (제안) 추천 리워드·할인 정책 → 입소문 가속 (원장 확인) |
| 2 | **블로그** — 거의 대부분 | 주기적 업로드 자동화 → [[블로그_자동화]] + 인스타 병행 |
| 3 | 입지·간판 워크인 | - |
| (검색) | **네이버 플레이스** "일곡동 영어학원" 상위노출 | 유지 = ① 학생 리뷰 독려 ② 블로그 주기 업로드 |

- **타깃**: 현재 유입 = 고1·2 중심 → **중등 비중 확대**가 목표.
- **셀링포인트**: **개인 완전 특화**(에이닷 핵심 메시지).
- **HAWE 전환**: 시점 미정(멀음). 단, 블로그 **입지는 지금부터 꾸준히** 쌓아 전환 대비.

## 리뷰·프로모션 & 콘텐츠 전략

### 프로모션 (뽑기) — 학습 동기 + 리뷰 독려
- 단어 100점 → **뽑기 1개** / **리뷰 작성 → 뽑기 5개**(리뷰 유도).
- 리뷰 독려는 결국 **사람이** 하는 일 → 프로모션 기획은 **조교와 함께** 구상 ([[학원 자원]] 인적 자원).

### 콘텐츠 전략 (에이전트 역할)
- **잘 나가는 학원 벤치마킹** + **유행 릴스·중고등 타깃 트렌드** 리서치 → 어떤 게시물 올릴지 기획.
- 채널 현황: 블로그 **불규칙**(1·2월 주3 → 3·4월 0 → 5월 2 → 6월 미게시) = 자동화 필요 → [[블로그_자동화]]. 인스타 **계정 有·미활동** → 릴스·이벤트 게시물로 시작.

## 담당 업무

- HAWE 브랜드 운영 → [[브랜드_디자인_가이드]]
- 블로그·SNS 홍보 콘텐츠 제작
- 신규 문의 응대 템플릿·흐름
- 브랜드 자산 관리

## 입력 → 산출 → 핸드오프

- 읽음: [[브랜드_디자인_가이드]](HAWE), [[톤앤매너]] 외부채널, 학원 성과·사례(민감정보 제외)
- 만듦: 홍보 콘텐츠, 신규문의 응대 템플릿, 브랜드 자산
- 넘김: → 원장(게시·발송 최종 승인) / → [[3_상담소통부]](문의 → 상담 전환)

## 핵심 규칙

- 과장·성적 보장 표현 금지, 동의 없는 학생 사례 사용 금지 → [[금지표현]]
- 5_시스템개발부가 만든 도구(단어앱·교실 등)를 홍보 자산으로 활용 가능 (엔진=개발부, 홍보 활용=브랜드홍보부, R1)

## 신규 유입 → 등록 흐름

첫 후킹(블로그·인터넷 검색·플레이스) 이후 두 경로:
1. 후킹 → **네이버 플레이스 예약** → 전화 → 레벨테스트·상담 날짜 확정 → 등록 여부 확인 → **OT**
2. 후킹 → **바로 전화** → 레벨테스트·상담 날짜 확정 → 등록 여부 확인 → **OT**

부서 연결: 후킹 = 6_브랜드홍보부 · 전화 응대·상담 날짜 = [[3_상담소통부]] · 레벨테스트 진단·배치 = [[2_교무부]]([[레벨테스트]]) · 등록 = [[4_행정운영부]] · **OT(오리엔테이션)** = (담당·내용 원장 확인)
