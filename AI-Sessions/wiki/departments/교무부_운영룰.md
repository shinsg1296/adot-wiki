# 교무부 운영 룰 (#1 진도 · #2 숙제 · #3 검사 · #6 자료)

[[2_교무부]] 기능 ②(배정·설계)·③(검사)의 실행 기준. **전제: 모든 수업·진도·숙제는 개인 특화**(반 아님). 트랙=정규·내신·수능.

## #1 진도 기준

- **신규생 배치**: 입학 시 **레벨 테스트**(듣기·어휘·구조·독해·언어력)로 진단 → 점수별 교재 배치(고듣끝·고단끝·독해 커리큘럼). 기준·매핑 → [[레벨테스트]].
- **개인별**. 조정 트리거: 실력차 · 목표 등급 · 학교 시험일 · 결석 따라잡기.
- 정규 = 학원 정규 커리큘럼(curriculum_toc) 순서.
- **진도 기록**: 현재 원장 머릿속 + CRM(강의 진행 시 몇 강인지 표시). 학생_마스터엔 배정 교재명만(몇 강까지는 없음) → *진도 단일 출처화는 개선 과제(머릿속 진도는 이식 불가).*
- **정규 → 내신 전환: 기본 5주 전**(개인차). 전환 시 정규 강의·단어 멈춤 → 시험범위 강의(강의 생략하고 숙제만 이행도 가능).
- 수업 빈도 = **주 1회**. 강의·단어는 집에서(주로 5일 분산, 학생이 수강 시간 지정) → 자동화 [[강의수강_카톡_자동화]].

## #2 숙제 표준 + 조정 기준표

### 정규 표준 (주간)
- 고등: 단어 5강(≈165개, 33×5) + 강의 5강
- 중등: 단어 3강 + 강의 3강
- 학년 구분은 관습, 실제 결정은 **실력 + 성실도**.

### 수능 트랙
- 18지문(20·21·22·23·24·29~40·41~42) 3지문×6일 + 리뷰테스트·단어 인증 → [[SOP_모의고사_숙제]]

### 조정 기준표 (원장 교정 2026-06-07 — 로직 검증 / 수치는 제안)
> 적용 로직은 원장 교정 완료, 구체 수치는 제안 단계. 리스크 추적 → [[디리스킹_장부]].

> 핵심: **이해 부족 ≠ 강의량 줄이기.** 강의량은 *스케줄·의지* 문제일 때만 조절한다.

| 신호 | 해석 | 조정 (지도법 우선) |
|---|---|---|
| 정규 연속 C / AI테스트(복습·리얼) 80% 미만 | **이해 부족**(강의 숙지 안 됨) — 강의량 문제 아님 | 3원인 진단 → 지도법 변경: ① 집중도=강의 재수강 ② 내용 난이도=암기·이해 자료 재제공 ③ 문제 이해력=해설 + [[오답진단_질문체인]] |
| 강의 미이행·지연 | **스케줄·의지** 문제 | 강의량 조절(스케줄·의지에 맞춤) + 계획 재조정 |
| 단어 재시험 반복 | 암기량 과다 가능 | 단어량↓ **또는 새 암기 방법 제시** |
| 정규 A 안정 + 계획 100% | 여유 | 강의·단어↑ 또는 심화(어휘끝 등) |
| 결석 | 진도 누락 | 따라잡기(분산 보충) |

- 원칙: 강의량 조절은 **스케줄·의지에만**, 이해 부족은 **지도법 변경**으로. 2주 관찰 후 재조정. 목표 등급 높을수록 기본기 우선([[내신_성적향상_로드맵_방법]] 난이도 모델 연계).

### 암기법 매칭 (단어 재시험 반복 시)
- "새 암기 방법" = 아이에게 맞는 암기법을 찾아주는 것: **깜지(쓰기) / 발음 / 경선식(연상법)** 등.
- **기본 점검**: 발음을 못한 채 암기하는 학생 = 암기 준비가 안 된 것 → **발음 먼저** 연습시킨다.

## #3 검사 · 재시험 기준

### 자동 / 수동
- 단어 = 단어앱 자동 / 모의 = 빠른채점 / 100발100중 자습서 = **조교 수동 채점**([[학원 자원]] 인적 자원).

### 재시험 기준 (현행)
- **강의별 복습 AI · 리얼 AI**: 15문항, 1개까지 허용(14↑ 통과), 2개↑ 재시험. **매 강의 통과 필수.**
- **정규 AI(누적)**: 주 1회 원장 수동 발송(1~5강 취약점), 20문항 5분, 2개까지(18↑ 통과), 3개↑ 재시험.
- **단어**: 30개, 3개까지(27↑ 통과), 4개↑ 재시험.

### 재시험·오답 솔루션 (단순 재시험에 그치지 말 것)
- 80% 미만/재시험 = **이해 부족 진단** → 원인별 지도(집중=재수강 / 내용=자료 재제공 / 문제 이해=해설).
- 오답은 [[오답진단_질문체인]]으로 막힌 지점을 찾아 **그 부분부터 재학습**.
- 현 AI테스트는 이 솔루션을 안 줌 → [[AI진단_문제풀이_솔루션]] 개발 예정(교무부 설계 → 개발부 구현).

### 미제출 · 미인증
- 학생 독려 + **학부모 알림**(카톡) → [[강의수강_카톡_자동화]] · [[숙제인증_앱]].
- 단어테스트 기록뷰어로 **미실시 집계**.
- AI테스트 시스템(복습/리얼/정규) = **도구는 [[5_시스템개발부]](R1)**, 운영·취약점 선정은 교무부/원장. (정규 AI **자동 발송** 가능 여부 추후 검토.)

## #6 자료 배정 룰

### 단어장
- 고등 기본 **고단끝** → 고3 + 암기 완료 = **어휘끝 어원편**
- 중등 기본 **중단끝** → 학교 단어장 사용자 = 학교 것(예: 100발100중)

### 강의(정규)
- 정규 커리큘럼(curriculum_toc).
- 중등 문법 정규 = **Grammar Ace 1/2 → Build-up**(레벨테스트 총점 배치 → [[레벨테스트]]). 중등 단어 = 중단끝.

### 내신 자료
- **고등**: 교과서 + **모의 4개년 기출**(1학기 중간=3월 · 1학기 기말=6월 · 2학기 중간=9월 · 2학기 기말=11월, *본인 학년* 4개년) + **학교별 부교재**(첫단추 독해유형편 · minute70 등).
- **중등**: 100발100중 + 에이닷 강의용교안.
- 부교재 매핑: 학기초 학생이 알려주면 매핑, **진도는 매번 학생에게 확인**(수동).
