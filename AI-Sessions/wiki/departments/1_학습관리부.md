# 1. 학습관리부 ⭐

**미션**: 매일 수업 결과를 기록하고 학부모에게 신뢰를 주는 수업일지·성적 관리

## 담당 업무

- 매일 밤 10시 CRM 수업일지 입력 (성적 2개 + 코멘트 4필드) → [[SOP_수업일지_작성]]
- 성적용 코멘트에 **학원 진도 ↔ 학교 교과서 진도 연결성** 녹이기
- 수업 녹음 전사 → 학생별 4필드 작성 워크플로우

## 핵심 규칙

- **작성 주체 = 각 담당 강사**(본인 학생 직접 작성). 톤·품질 통일 = [[톤앤매너]]·[[SOP_수업일지_작성]] 공유(강사 공통 규칙).
- 4필드: 성적용 코멘트(학부모 톤) / 면담용 특이사항 / 원장 솔루션 / 주간 숙제범위
- 톤: "인상적이었습니다" ❌ → "기특해서 칭찬해주었습니다" ⭕ (선생님 능동 시점)
- 입력 후 재확인 없이 바로 **작성 완료** 클릭 (임시 저장 X)
- 진도 의미는 curriculum_toc + lesson_progress_map 항상 참조

## 수업 1회 구조 (주 1회 대면)

집에서 강의·암기 → 대면 1회는 **점검·피드백·심화**:
1. 지난 한 주 점검
2. 단어 누적 테스트 + 정규 AI 결과 **피드백**
3. 정규 AI 후 **심화 응용 테스트(구두)** — 원장이 묻고 **학생이 직접 설명** → 이해도 판단 ([[오답진단_질문체인]]의 대면판)
4. 판단 후: 부족=공부 전략·해결책 제시 / 게으름=정신교육 / 잘함=칭찬

## 성적 산출 기준

- **단어**: 30개 테스트 점수 그대로(30점 만점). 재시험 컷 → [[교무부_운영룰]] #3.
- **정규 A·B·C** (정규 AI 통과 후 **구두 테스트** 틀린 개수로 결정):
  - 정규 AI = 20문항, **18↑ 통과 / 17개(3틀림)↓ = 재시험**.
  - 통과 시 구두 테스트 틀린 수: **1 → A · 2 → B · 3 → C(C면 재시험)**.
  - 정규 AI에서 17개로 재시험이면 → 구두 없이 **바로 C + 재시험**.
- 구두 테스트 = 위 '수업 1회 구조'의 심화 응용 구두 ([[오답진단_질문체인]] 대면판).
