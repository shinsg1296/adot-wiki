# 명령어 사용법 (사람용 요약)

에이전트에게 일을 시킬 때 쓰는 **영어 키워드 4개**입니다. 키워드는 영어로 고정, 자연어로 말해도 됩니다.

| 키워드 | 자연어 예시 | 한 줄 설명 |
|---|---|---|
| `save` | "이번 작업 내용 옵시디언에 저장해줘" | 이번 작업 맥락을 위키에 저장(5필터 통과 시) |
| `query` | "옵시디언 참조", "옵시디언 참조해줘" | 위키·로그에서 이전 맥락 복원 |
| `ingest` | "이 자료 ingest 해줘" | `raw/` 원본을 읽어 위키 문서로 가공 |
| `lint` | "위키 점검해줘", "lint 해줘" | 위키의 모순·중복·출처·낡은 규칙 점검 |

## 매 세션 사용법

- 세션 **시작**: "옵시디언 참조" (`query`) → 이전 맥락을 먼저 복원.
- 세션 **끝**: "이번 작업 내용 옵시디언에 저장해줘" (`save`) → 맥락 이어가기.

## 안전장치 (요약)

- `save`는 **5가지 저장 필터**를 통과한 것만 저장합니다(맥락 오염 방지).
- `raw/` 원본은 **절대 수정하지 않습니다.**
- 검증 안 된 추측·일회성 답변은 위키에 올리지 않고 `conversations/`까지만 둡니다.

> 정확한 read/write 규칙과 안전장치 전체는 루트 **[[CLAUDE]]**의 "§3 명령어 계약(Command Contract)" 표를 단일 기준으로 따릅니다.
