# 리뷰 기준 (완료 정의)

이 기준을 통과하기 전에는 "완료"라고 보고하지 않습니다.

## 공통 (Definition of Done)

- [ ] **작동 증명**이 있는가? (결과·로그·스크린샷으로 확인) — 추정 금지.
- [ ] "**숙련된 담당자가 승인할 수준**인가?" 자문 통과.
- [ ] 변경이 **필요한 곳만** 건드렸는가(최소 영향).
- [ ] 민감정보·개인정보가 노출되지 않았는가. → [[금지표현]]
- [ ] 저장이 필요하면 5가지 필터 통과 후 저장했는가. → [[명령어_사용법]]

## 업무별

| 업무 | 추가 검수 |
|---|---|
| CRM 입력(수업일지 등) | 저장 후 **결과 재확인** 필수 → [[SOP_수업일지_작성]] |
| 출력물 | 컬러/흑백·동반 출력 규칙 확인 → [[SOP_내신자료_출력]] |
| 모의고사 자료/레이아웃 | 골든 레이아웃 검수 통과(`verify_layout.py` → 골든통과) |
| 위키 문서 | `lint` 통과(모순·중복·출처·stale 점검) → [[명령어_사용법]] |

## 교정 후 학습

- 사용자에게 교정받으면 패턴을 `tasks/lessons.md`(또는 `wiki/errors/` 문서)에 남겨 재발을 막습니다.
