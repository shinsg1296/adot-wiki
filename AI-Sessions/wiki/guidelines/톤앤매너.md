# 톤앤매너 (채널별)

학원의 모든 글은 **신뢰**를 만드는 도구입니다. 채널마다 받는 사람이 다르므로 말투를 구분합니다.

## 채널별 기준

| 채널 | 대상 | 말투 | 핵심 태도 |
|---|---|---|---|
| 수업일지·알림톡·상담 | 학부모 | 경어 | 신뢰·구체적 사례·선생님 능동 시점 |
| 수업·숙제 피드백 | 학생 | 평어 | 격려·구체적 개선 방향 |
| 앱 UI (단어앱·빠른채점·교실) | 학생 | 평어(간결) | 깔끔·따뜻·군더더기 없음 |
| 외부·마케팅 | 학부모/일반 | 경어 | 브랜드 철학([[브랜드_디자인_가이드]]) 반영 |

## 학부모 채널

- 항상 **경어**. 평가는 **선생님의 능동적 행위 시점**으로 씁니다.
  - ⭕ "기특해서 칭찬해주었습니다" / "한 번 더 짚어주었습니다"
  - ❌ "인상적이었습니다"(관찰자 톤) — 자세한 금지 표현은 [[금지표현]]
- 학원 진도와 **학교 교과서 진도의 연결성**을 녹입니다. → [[SOP_수업일지_작성]]
- 사례 없는 칭찬·단정 대신 **구체적 관찰**로 신뢰를 만듭니다. → [[3_상담소통부]]

## 학생 채널

- **평어**, 격려 중심. 틀린 것은 비난이 아니라 **다음 행동**으로 연결합니다.
- 성향에 맞춰 조정합니다(예: 토론·논리형 성향(MBTI ENTP 등)은 논쟁 유도형 질문). 학생 개별 코칭은 해당 학생 노트(`wiki/students/`) 참조.

## 앱 UI 채널

- 짧고 명확하게. **이모지 남용 금지**, 라인 아이콘·차분한 톤(현재 테라코타, 전환 검토 → [[브랜드_디자인_가이드]]).
- 안내 문구는 [[나레이션지침]]을 따릅니다.

## 외부·마케팅 채널

- 브랜드 철학("정답을 바로 가르치지 않고 함께 경험·질문하며 진화")을 반영합니다.
- 과장·성적 보장 표현 금지 → [[금지표현]].
