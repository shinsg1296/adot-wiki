# 평가원 10유형 출제원리 (통합)

## Summary

[[하위큐_문제생성_시스템]]이 생성하는 **10유형의 출제원리를 한 곳에** 정리. 각 유형의 **정답 설계 원리 + 오답 패턴 + 정답 위치/특이점**. 제목·5유형은 [[제목추론_출제원리]]·[[수능_5유형_출제원리]]에 전수 분석(A1~A6·B1~B6 등) 상세가 있고, 여기선 통합 요지 + 신규 4유형(주제·요약·함의추론·어법) 핵심을 담는다.

## Details

### 공통 원리 (모든 유형 관통)
- **정답 = 글 전체 논지/방향과 정합** · **오답 = 소재어는 비슷하되 방향·범위·관계에서 벗어남**.
- 오답 5대 골격: **부분(범위과소) / 과대 / 방향역전 / 날조·무관 / 관계왜곡**. + 소재어 재사용으로 그럴듯하게.
- **복수정답 차단**: 오답은 "뒤집으면/바꾸면 즉시 어색"하게 설계.

### A. 핵심 6유형

| 유형 | 정답 원리 | 오답 핵심 | 위치/특이점 |
|---|---|---|---|
| **제목** | 글 전체 범위 포괄, 함축·은유 가능(콜론 2부구조) | 부분·과대·역전·날조·극단어 | 상세 [[제목추론_출제원리]] A1~A6/B1~B6 |
| **어휘** | 결론부 **극성 내용어를 반의어 교체**=정답, 나머지 정방향 | 반의어 아닌 정방향(뒤집으면 어색) | 정답 **④·⑤**(①~③ 0회). 해설에 [접근:주제대조형/문장논리형] |
| **빈칸** | **재진술 짝**(일반화 문장 ↔ 근거)을 패러프레이즈 | 극성역전·소재어 관계왜곡·미언급 | 선지 5개·통념 함정 |
| **무관** | 직전 **소재어 재사용**한 이탈 문장, 빼면 앞뒤 직결 | (정답=무관문장 자체) | 위치 **③·④**, 새 명사=막다른 길 |
| **순서** | 연결어·지시어 **단서 절단**, given→정렬 | A-first 회피, 지시어 받침 끊김 | 정답 ②~⑤(① 회피) |
| **삽입** | **경첩문장**(앞 받고 뒤로 넘김)+박스문장, 받침 | 받침 없는 위치 | 위치 **③·④·⑤** |

(상세 = [[수능_5유형_출제원리]])

### B. 신규 4유형 (2026-06-12 실물 분석)

**주제** — 선지 구조 = `[내용 추상명사 head] + of/관계사 + [소재(+수식)]`
- 내용어 **극성**(＋necessity/중립role·way·factors/－challenges)·**단복수**(복수=나열형/단수=단일초점)로 **글을 역추론**.
- 정답: 내용어 극성=글 stance·단복수=글 구조 일치 + 패러프레이즈.
- 오답: 부분진술(소재만 맞고 내용 약함)·과대·극성역전·날조·단복수불일치.
- **해설 필수**: 오답마다 **틀린 단어 빨간 취소선 + (반대/무관) 라벨**.

**요약** — 한 문장 요약 + 빈칸 (A)(B) → 5선지 각 (A)/(B) 쌍
- 정답: 두 핵심요소 패러프레이즈 + 논리관계(인과·영향·대조) 완성, **둘 다 정확**.
- 오답 핵심: **★한쪽만 맞는 함정**(정답 칸을 여러 선지가 공유 → 한 칸만 보고 고르게 유도) · 극성역전 · 유사어함정 · 관계왜곡(B동사).
- **반드시 두 칸 다 검증.** 적합도: 논지·인과 뚜렷한 글만(단순나열 skip).

**함의추론** — **선별 출제**(비유·관용·역설 핵심구 있는 지문만, 부적합 skip)
- 수능 23~26 밑줄구 전부 은유: *nonstick frying pan / hunting the shadow not the substance / made work less sticky / make oneself public to oneself*.
- 정답 = **(밑줄 구 속뜻) ∩ (글 주제 방향)**: 표면뜻✗ 문맥속뜻✓, 앞뒤 설명+논지 직결, 패러프레이즈.
- 오답 6패턴: 표면뜻 오독·방향역전·관계왜곡·부분진술·무관날조·극단화(never/only/any) + **지문 소재어 재사용**.

**어법** — 구조형 : 해석형 ≈ **4:6**(모의고사 트렌드=해석형)
- 구조형(형태만): 정동사/준동사·기본 수일치·형/부·기본 관계사.
- **해석형(의미 파악 필수, 트렌드)**: 대동사(앞동사 의미·태·수)·대명사 지칭(선행사 의미)·능동/수동(주어가 하나 당하나)·**수식어구 분리 후 수일치**(멀리vs가까이 의미판단)·의미기반 병렬(누구와 병렬인지 맥락)·관계사.
- 설계: 틀린 곳 1개 **명백·유일**, 나머지 4개 확실히 맞음. 해석형은 "형태만으론 안 보이고 의미 읽어야 잡히게". **강한 문법검증 게이트 필수**.
- 해설: 틀린 곳 규칙위반+올바른형태(해석형은 "무슨 의미 때문에 이 형태"), 나머지 4개 각 포인트+왜 맞는지.
- ★어법 = 지문변형이 쓰는 문법(분사구문·도치·태·관계사·수일치)을 **거꾸로 한 곳 틀리게** → 변형엔진과 같은 지식 공유.

### 지문 변형 (출제와 짝)
원문 암기 방지 + 학교 변형 모방. **L2 적극 구문변형**(태·강조·도치·부사절→분사구문·구↔절) = 주제/제목/무관/빈칸/순서/삽입, **L1 경미** = 어법/어휘. 의미·문법정확·정답근거 신호어 보존. 상세 → `_haweq_next_spec.md`.
