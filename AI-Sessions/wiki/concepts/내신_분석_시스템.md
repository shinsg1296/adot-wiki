# 내신 분석 시스템 (시험지 분석 · 출처 매칭 · 출제 프로파일)

> **후속 아키텍처(2026-07)**: 이 문서는 초기 `analysis/`·db_corpus 매칭 구조다. 현재는 `C:\시험지분석\master\`(원문뱅크 + 인터랙티브 웹 분석지)로 발전했고, 제작 규칙은 → [[SOP_내신분석지_제작법칙]].

광주·전남권 영어 내신 **기출 시험지를 데이터화**하고, **지문 출처를 자동 식별**하며, 학교별 **출제 프로파일**(다음 시험 예측·대비)을 만드는 시스템. 교무부 ④시험지분석·⑤예상문제([[내신_성적향상_로드맵_방법]])의 실제 구현체이고, 위키 백로그 ⭐[[내신_자동생성_파이프라인]]의 핵심 절반에 해당.

> **데이터는 위키가 아니라 파일에 있다.** 위키엔 "무엇이 어디 있고 어떻게 쓰는가"만 둔다(대용량 지문·JSON은 코드/파일이 다룸). 분류 근거 → [[이식성_체크리스트]] (데이터층/지식층 분리).

## 위치 (데이터층)

- 루트: `C:\시험지분석\`
  - `extracted\` — 압축 해제 원본 PDF
  - `pages\<시험>\pNN.png` — 전 페이지 렌더 이미지 (비전 분석용)
  - `analysis\<시험>.json` — **시험별 분석 결과 241개** (문항·유형·배점·출처·서술형·정답·손글씨)
  - `passages\<시험>.json` — 시험별 영어 지문(매칭용)
  - `report\` — `00_전체요약.md` · `01_원장_종합보고서.md` · `02_출처복원_결과.md` · `master_table.csv` · `학교별\<학교>.md`(출제 프로파일 36개)
  - `db_corpus.json` — **매칭 코퍼스 (지문 약 27,000개)**: 학평·평가원·수능(21~26)·EBS 수능특강 + 모아닷 부교재 + 로컬 부교재

## 파이프라인 (스크립트)

| 단계 | 스크립트 | 하는 일 |
|---|---|---|
| 해제·인벤토리 | `unzip_and_inventory.py` | zip 해제(CP949 안전) + 목록 |
| 렌더 | `render_all.py` | 전 페이지 PNG |
| **시험 분석** | `wf_extract.js`(Workflow) | 페이지 비전 분석 → `analysis/*.json` |
| 집계 | `aggregate.py` | master_table·요약·school_stats |
| **출제 프로파일** | `wf_synth.js`(Workflow) | 학교별 프로파일 md |
| 코퍼스 구축 | `build_db_corpus.py`(모의고사 시트) · `build_moadat_corpus.py` · `build_share_xlsx_corpus.py` · `build_local_bugyojae_corpus.py` | 지문+출처를 `db_corpus.json`에 병합(중복제거) |
| **출처 매칭** | `match_passages.py` | 시험 지문 ↔ 코퍼스 토큰 매칭(불용어 제거, threshold 0.6) |
| 출처 복원 | `integrate_sources.py` | 매칭으로 analysis JSON '미상' → 실제 출처 갱신 + 복원 리포트 |

## 현재 성과 (2026-06-09)

- 시험 **241개** 전수 분석(손상 0), 학교 **36개** 프로파일.
- **중등 = 교과서 83%** / **고등 = 외부지문(모의·수특·부교재) 중심.**
- 출처 매칭: 고등 '미상' **32% → 17%** (약 700문항 출처 복원). 코퍼스 27,000개.
- 학교 유형 4분류(교과서형/모의고사형/수능특강형/혼합·고난도형) → 종합 보고서(`C:\시험지분석\report\01_원장_종합보고서.md`).

## 에이전트 사용법

- **새 시험지 분석**: zip→`unzip_and_inventory`→`render_all`→`wf_extract`→`aggregate`(필요시 `wf_synth`).
- **지문 출처 찾기**: 지문 추출→`match_passages.py`(코퍼스 대조). 손글씨 메모가 1차 근거.
- **코퍼스 키우기**: 새 자료(합법 보유분)를 `build_*_corpus`로 병합(중복제거). 새 출처일수록 미상 감소 효과 큼(중복은 효과 미미 — 데이터로 확인됨).

## 한계 · 확장

- 남은 미상(17%) = 사설 모의(이감·바탕·더프)·수능완성·**학교 자체 변형 지문**. → 공유폴더2(`\\desktop-07gd04s\일곡점 공유폴더2`)의 족보닷컴·사설 자료 ingest로 더 감소(공유 PC 켜져 있어야 접근).
- 자체 변형 지문 일부는 원본이 없어 **본질적으로 미상**일 수 있음.
- **변형 분석(다음 단계)**: 매칭된 지문은 원본 대비 diff로 "어디가 변형됐나"(어휘 교체·어법 변형·어순 등) 식별 가능 → 출제 스타일·정답 포인트 예측.
