# 이식성 체크리스트 (다음 사업으로 무엇을 들고 가나)

[[0_본사_전략실]] ③의 산출물. 에이닷에서 쌓은 자산을 **이식층(복사 가능)** vs **도메인층(에이닷 전용)** 으로 분류한다. 진짜 산출물 = ① 이식 플레이북 ② 성장한 운영자. (근거: [[의사결정_조직구조]])

## 이식층 — 다음 사업으로 복사 ✅

| 자산 | 위치 | 상태 |
|---|---|---|
| 명령 체계(save/ingest/query/lint) + 저장 안전장치 | [[CLAUDE]] | 검증됨 |
| 가이드라인 7종(톤앤매너·금지표현·나레이션·기획·리뷰·명령어사용법·개요) | `wiki/guidelines/` | 검증됨 |
| 부서(역할) 모델 + 핸드오프 + 분담규칙 R1~R3 | [[부서 개요]]·[[의사결정_조직구조]] | 검증됨 |
| 성적향상 로드맵 방법(난이도=대비가능성·전략 매트릭스·양식 A/B) | [[내신_성적향상_로드맵_방법]] | 방법 검증(수치 일부 제안) |
| 레벨테스트 진단·배치 프레임 | [[레벨테스트]] | 기준 검증(교재 매핑은 도메인) |
| 검사·재시험·성적 산출 로직 | [[교무부_운영룰]]·[[1_학습관리부]] | 검증됨 |
| 개발 자동화 패턴(백로그 7건) | `wiki/dev-tasks/` | 설계됨(미구현) |

## 도메인층 — 에이닷 전용(교체 대상) 🏠

- SOP(수업일지·내신자료·모의숙제·통화) · 학생 노트 · 학생_마스터 · 학교-출판사 매핑 · 모아닷 DB · CRM(classPlannerManager)·출결(아이패드)·홈페이지 · 인적 자원(강사·조교) · HAWE 브랜드.

## 경계 원칙

- 도메인층이 이식층에 새어들면 다음 사업 복사 시 **오염** → 분리 유지.
- 리스크·미검증 항목은 [[디리스킹_장부]]에서 추적.
