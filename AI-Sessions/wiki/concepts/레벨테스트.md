# 레벨테스트 (신규생 진단·배치)

신규생 입학 시 실력·취약점을 진단해 **트랙·진도·교재를 배치**하는 기준. 두 종류: **고등(수능용)** · **중등(그래머에이스)**. 파일·음원 위치 → [[참고자료 색인]].

신규 유입 루프의 마지막 칸: [[6_브랜드홍보부]] → [[3_상담소통부]] → [[4_행정운영부]] 등록 → **레벨테스트 배치**. 자동화 입력의 출발점 → [[내신_자동생성_파이프라인]].

---

## 고등 (수능용)

### 구성 (총 100점)
| 영역 | 문항 | 배점 | 만점 |
|---|---|---|---|
| 듣기 | 1~5 | 2 | 10 |
| 어휘력 | 6~35 | 1 | 30 |
| 구조력 | 36~50 | 2 | 30 |
| 독해력 | 51~65 (5×1 + 10×2) | 1·2 | 25 |
| 언어력 | 66~70 | 1 | 5 |

- 어휘 구간: 초급(중학) 6~10 · 중급(고1·2) 11~15 · 준고급(고3) 16~30 · 고급 31~35.

### 판정 → 처방
- **듣기**: 상/중 = 고듣끝 실전편 · 하(0~4) = 고듣끝 유형편
- **어휘**(고단끝 코스): 상 26~30 → 2개월 / 중상 21~25 → 3개월 / 중 11~20 → 6개월 / 중하 6~10 → 중학 30일 / 하 0~5 → 중학 60일
- **독해 커리큘럼**(구조+독해+언어 합산 60점): Intermediate 51~60(고3 모의 1~3등급) · Basic 26~50 · Build-Up 11~25 · Start-Up 0~10
- 구조 단독: 상 21~30·중 11~20·하 0~10. 언어력: 상 4~5·하 0~3(하 = 국어 언어력부터).

## 중등 (그래머에이스)

### 구성 (총 100점)
| 영역 | 만점 |
|---|---|
| 듣기 | 5 |
| 어휘력 | 30 |
| 구조력 | 20 |
| **문법** | 14 |
| 독해력 | 23 |
| 언어력 | 8 |

- 고등과 달리 **문법 영역**이 따로 있음 (중등 내신 = 문법 중심). 어휘는 뜻쓰기.

### 최종 커리큘럼 (총점)
| 레벨 | 총점 |
|---|---|
| Grammar Ace 1 | 0~40 |
| Grammar Ace 2 | 41~80 |
| Build-up | 81~100 |

- 단어 교재 = 중단끝 ([[교무부_운영룰]] #6).

### 영역별 단계 (참고)
어휘 최상 26~30·상 21~25·중 11~20·하 0~10 / 문법·구조 상 17~20·중 8~16·하 0~7 / 독해 상 26~30·중 11~25·하 0~10 / 듣기 상 4~5·하 0~3 / 언어력 상 7~8·하 0~6.
> 원본 엑셀의 영역별 등급 구간 일부가 만점과 어긋남(템플릿 스케일) — **총점 → 커리큘럼** 매핑을 우선 기준으로.
