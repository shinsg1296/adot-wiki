# [개발] 자동 수업 기록 (무조작 녹음·전사 → 수업일지)

담당: [[5_시스템개발부]] (요구: [[1_학습관리부]]) · 상태: **백로그(미착수)** · *4필드 병목 직격 → 우선순위 높음*

## 배경

4필드 작성이 손에 잘 안 잡히고 어렵다. 게다가 녹음·전사 자동화가 안 돼, 수업마다 **녹음을 켜고/끄는 조작**이 번거롭고 **깜빡할 변수**가 머리를 혼란스럽게 한다. → **"신경 안 써도 수업이 자동으로 적히게"** 가 목표.

## 요구 기능

- **무조작 녹음**: 수업 시작/종료를 자동 감지해 녹음(버튼 X). 트리거 = 시간표 기반? 입실 감지?
- **자동 전사**: 학생 이름 호명으로 화자 구분.
- **4필드 초안 자동 작성**: 학생별 성적용·면담용·솔루션·숙제범위 초안 → **강사 검수 후** CRM(classPlannerManager) 입력.

## 규칙

- 톤 = 선생님 능동 시점 경어 ([[톤앤매너]] · [[SOP_수업일지_작성]]). 최종 입력·검수 = **담당 강사**.
- 녹음 동의·보안 유의. 수업 녹음 폴더는 통화녹음과 분리 ([[SOP_통화기록_처리]]).
