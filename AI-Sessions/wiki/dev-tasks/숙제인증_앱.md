# [개발] 숙제 사진 인증 앱

담당: [[5_시스템개발부]] (요구: [[2_교무부]] 검사) · 상태: **✅ 배포 완료** → 운영 노드 [[숙제관리_CRM]] (이 백로그는 거기서 해소)

## 배경

강의를 안 듣고 숙제만 이행하는 경우가 있어, **강의 수강 여부와 무관하게** 학생이 숙제를 실제로 했는지 판단할 수단이 필요.

## 요구 기능

- 학생이 **숙제를 직접 찍어 인증 업로드**.
- 인증 기록으로 "강의를 잘 들었는지 / 숙제를 이행했는지" 판단 보조.

## 연결

- 검사(#3) **미제출·미인증 자동 추적**과 연동(누가 안 했는지 집계).
- 단어=단어앱 / 모의=빠른채점 / 자습서=조교 수동 채점과는 별개의 **'이행 인증' 레이어**.

## ✅ 완료 (2026-06-10 배포)

이 백로그는 **숙제관리 CRM 앱**으로 구현·배포되어 해소되었다. 기존 '미정(원장 확인)' 항목도 함께 결정됨:

- **앱 형태**: 단어앱과 **별도 앱**(프론트=Netlify, 백엔드=별도 Apps Script).
- **인증 방식**: 학생이 숙제 **사진 업로드** → 선생님 **승인/반려**(OCR 아님, 사람 검수).
- **미인증 추적**: 선생님이 학생별 숙제 배정 → 제출/승인 상태로 미이행 집계.
- 기록=`채점기록` 시트 `숙제관리`탭, 사진=Drive 폴더. 학생 로그인·선생님 게이트 분리.
- 2026-06-10 **시험범위 대시보드**(학교별 범위·학생 개인진도 3단계)까지 확장 배포.

> 코드·시트 ID·배포 URL 등 민감/운영 상세는 위키 본문에 미기재(원장 운영 메모리 참조).
