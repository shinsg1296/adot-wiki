# [개발 ⭐1순위] 내신 자동 생성 파이프라인

담당: [[5_시스템개발부]] (요구·검수: [[2_교무부]]) · 상태: **백로그(최종 목표)**

## 목표 (원장 최대 병목 해소)

- **입력**: 학생 정보(실력·취약점, [[레벨테스트]]·성적) + 학교별 기출 + 시험 범위
- **출력**: ① 학생별 **내신 계획**(주차별) + ② 학교별 **예비고사(예상문제) 세트**
- → 지금 원장이 가장 오래 걸리는 일([[기획기준]] 최우선 병목)을 자동화.

## 구성 요소 (이미 설계됨 — 통합이 과제)

- ④ 학교 시험지 분석 → 출제 프로파일 ([[내신_성적향상_로드맵_방법]] · [[템플릿_학교출제프로파일]])
- 계획 수립 ([[교무부_운영룰]] #1·#2)
- ⑤ 예상문제 생성 ([[내신_성적향상_로드맵_방법]] ⑤)
- 오답 진단 ([[AI진단_문제풀이_솔루션]])
- 엔진: 모의고사 자동화 · 답지생성 · exam_server · 골든레이아웃 (R1)

## 핸드오프 (에이전트 간)

- 교무부: 학생 진단·프로파일·출제 설계·검수.
- 개발부: 분석·생성·조판 엔진 구현·통합.
