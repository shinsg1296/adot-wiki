# [개발 ⭐1순위] 내신 자동 생성 파이프라인

담당: [[5_시스템개발부]] (요구·검수: [[2_교무부]]) · 상태: **백로그(최종 목표)**

## 1. 목표 (원장 최대 병목 해소)

원장의 최대 병목 = **학생별 내신 계획 수립 + 예상문제 세트 제작**([[기획기준]] 최우선 병목). 이 둘을 자동화.
- **입력**: 학생 정보(실력·취약점, [[레벨테스트]]·성적) + 학교별 기출 + 시험 범위
- **출력**: ① 학생별 **내신 계획**(주차별) + ② 학교별 **예상문제 세트**

## 2. 입/출력 정의

### 입력
| 항목 | 소스 | 비고 |
|---|---|---|
| 학생 기본정보 | 학생_마스터.xlsx | 이름·학교·학년·목표·배정교재 |
| 학생 실력 | 레벨테스트 결과 | 듣기·어휘·구조·독해·언어력 |
| 학교별 기출 | raw/(학교)/(년도).pdf | 3개년 필수(없으면 프로파일 불가) |
| 시험 범위 | 원장/강사 입력 | 교과서 단원·부교재 범위 |
| 출제 프로파일 | [[내신_분석_시스템]] | 기출 분석 산출물 |
| 문제 DB | [[문제DB_공유_데이터베이스]] | 지문 풀·유형 태깅 |

### 출력
| 항목 | 형식 | 비고 |
|---|---|---|
| 학생별 내신 계획(주차) | PDF/MD | 진도·숙제·예상문제 분량 |
| 학교별 예상문제 세트 | PDF(조판) | 골든레이아웃(메모리 rule_golden_layout), 학원 내부용 |
| 출제 프로파일 갱신 | MD | 시즌마다 |

## 3. 파이프라인 단계

```
[입력 수집] → [기출 분석 ④] → [출제 프로파일] → [학생 진단] → [로드맵 ⑤] → [예상문제 생성 ⑤] → [조판] → [검수] → [배포]
```

1. **입력 수집** — raw/ 기출 ingest + 학생_마스터 읽기. (TODO: 수동 vs 자동 경계)
2. **기출 분석 ④** — PDF→유형분포·범위·변별 태깅, 난이도=대비가능성([[내신_성적향상_로드맵_방법]]), 산출=[[템플릿_학교출제프로파일]] 양식 A
3. **출제 프로파일** — 학교×학년별, 암기커버%·기본기%·변별%
4. **학생 진단** — 레테+성적+목표. 두 레버=기본기 수준×암기 투자량
5. **로드맵** — 전략 매트릭스→목표×시험성격→경로. 산출=[[템플릿_성적향상로드맵]] 양식 B
6. **예상문제 생성 ⑤** — 교과서 변형 + DB 유사 외부지문, 유형분포 맞춰 세트. 저작권=학원 내부전용. ([[하위큐_문제생성_시스템]]·[[하위파이널_내신제작]] 엔진 활용)
7. **조판** — 골든레이아웃, 강의용교안(컬러)+리뷰테스트(흑백)
8. **검수 ⭐ 자동/수동 경계=원장 확인** — 제안: 분석·생성·조판=자동 / 로드맵·출제방향·최종배포=원장 검수
9. **배포** — 공유폴더/출력 → [[4_행정운영부]]
