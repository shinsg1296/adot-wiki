# [개발] 블로그 자동화 (+ 인스타·플레이스 입지)

담당: [[5_시스템개발부]] (요구: [[6_브랜드홍보부]]) · 상태: **백로그(미착수)**

## 배경

신규 유입 2위 = **블로그**(거의 대부분). 네이버 플레이스 상위노출 유지에도 블로그 주기 업로드가 핵심. 매일 쓰는 게 일이라 자동화 대상. 현재 운영이 **불규칙**(1·2월 주3 → 3·4월 0 → 5월 2 → 6월 미게시)이라 꾸준함 확보가 목표. 인스타는 계정만 있고 미활동.

## 요구 기능

- **시험 분석 블로그 자동 생성·업로드**: 시험(예: 6모) 끝나면 빠르게 분석 → 블로그 글 자동 작성·업로드.
- **학생 학원 생활 콘텐츠**: 주기적으로 찍은 사진·활동을 블로그·인스타 글 소재로.
- **인스타 병행 운영** (릴스·이벤트 게시물).
- **콘텐츠 전략 보조**: 잘 나가는 학원 벤치마킹 + 유행 릴스·중고등 트렌드 타깃 → 게시물 주제 기획.
- 네이버 플레이스 상위노출 보조: 블로그 주기성 + (학생) **리뷰 독려** 흐름.

## 규칙

- 외부 채널 톤 = 경어·브랜드 철학, 과장·성적보장·**동의 없는 학생 사진/사례 금지** → [[금지표현]] · [[톤앤매너]].
- 게시 최종 승인 = 원장. 시험 분석 소재는 [[2_교무부]] ④ 분석과 연계.
- HAWE 전환 대비해 **지금부터 꾸준히** 누적(입지 축적).

## 미정 (원장 확인)

- 블로그 플랫폼(네이버?), 자동 업로드 방식(API/반자동), 학생 사진 동의 절차, 발행 주기.
