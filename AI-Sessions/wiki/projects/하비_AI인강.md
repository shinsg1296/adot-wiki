# 하비 AI 인강 (HAWE 강의 제너레이터)

## Summary

- **쿼카 마스코트 '하비'가 진행하는 AI 인강** — 위화감 없이 학생이 집중하는 강의. 핵심 통찰: 위화감은 얼굴이 아니라 '대본 말투'에서 옴 → 순서=대본→음성→얼굴.
- **제너레이터 완성(2026-07-19)**: 지문 텍스트 → 분석(claude -p 구독제) → 대본·장면 → TTS(ElevenLabs Deoksu) → 조립 → 완성 강의 HTML. 전 과정 자동, 편당 원가 ~2천 원.
- 미니모의5(첫단추 독해유형편) **17편 배치 생산** 진행(2026-07-20 기준 13편+ 완성), 학생용 허브(index.html) 운영.
- HAWE 사업의 킬러 상품이자, 학교 페이지 무료 콘텐츠의 원천.

## Context

수동으로 1편(브레인스토밍)을 원장 피드백 루프로 완성 → 그 패턴을 사양으로 제너레이터 구축. **엔진은 완성돼 있고 제너레이터는 데이터(JSON)만 생성**하는 구조. 파일럿(2번 "책 먼저? 영화 먼저?")에서 원장 지적(장면 재사용·구체 예시 부재·TTS 잘림 등)을 전부 파이프라인에 반영.

## Details

**강의 구성(4트랙, 학생이 골라 듣기)**: 내용 이해(개념 각인 장면·통념→반전→이유 장면들·주제·주제 문제) / 순서·흐름(구조 뼈대 다이어그램·직접 배열·연결사/지시어 신호 세그·흐름 문제) / 해석(문장별 직독직해=말 따라 하이라이트+원리 바+관계 화살표) / 문법·서술형(어법 출제 포인트·서술형 유형·어법 문제). 인터랙티브 문제=단일 선택·순서 배열(취소/채점하기)·밑줄 어법.

**제작 원칙(원장 확정)**: 대본=군더더기 금지·기본 문법 개념 설명 금지·공감 훅(조별과제·해리포터 등 실존 사례)·괄호/물결 금지(TTS 잘림) · 내용 이해 파트에서 영어 표현 언급 금지(지문 보기 전) · 어법=그 지문에서 실제 헷갈릴 것만+대동사·분리수일치는 존재 시 필수 · 서술형=나열 글→이유 나열형, 영작=특수구문만 · 장면=compose DSL(부품 조립: 책·클래퍼보드·인물·말풍선 등, 실존 유명 사례 라벨) · 학교 간 비교식 표현 금지.

**품질 게이트**: ①강의 깊이 표준(8차원 채점표, 원장-AI 1차 보정 완료: AI가 ~1레벨 후함, 구조 노출이 최대 맹점→A뼈대+B배열로 확정) ②TTS 잘림 자동 검수(파형 끝 진폭 스캔→검증-재시도 합성) ③대본 JSON 자가 수리 ④정답=공식·검증 원칙.

**기술 스택**: `gen/analyze.py`(분석)·`gen/script_gen.py`(대본)·`gen/assemble.py`(TTS+조립)·`gen/batch_mm5.py`(배치)·`_mt5_page2.py`(빌더, 인자화). 음성=**Deoksu 확정**(클론은 추가 옵션, 교체 없음→재TTS 리스크 0). ElevenLabs Creator(121k자/월), 월 100편 시 Pro 전환.

**저작권 스탠스**: 첫단추 기반 강의=재원생 제공+무료 홍보까지(쎄듀 문의 회신 대기). 판매 라인=기출·자체 지문.
