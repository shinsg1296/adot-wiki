# 동신여고2 파이널고사 (영어Ⅰ · 26년 1학기 1차 대비)

동신여고2 **26년 1학기 중간 비중·출처를 역설계**해, 시험범위에서 변형·생성한 **파이널 1·2·3차(문제지+답지)** 를 정식 학교 공식폼으로 제작. 변형 엔진 = [[하위큐_문제생성_시스템]] 뱅크 + 생성, 분석 관점 = [[서술형_출제원리_분석]].

## Summary

- 26중간 = **27객관식(80점) + 4서술(20점) = 100점**. (학교 표지 템플릿의 "21문항" 표기는 구버전 — 실측 27객관식)
- 파이널 3차 각각 31문항. **출처 카테고리는 26중간 그대로 매칭**(수특칸→수특, 모의칸→모의), 교과서는 객관식 1문제만.
- 산출물: 문제지 3 + 답지(정답·출처·해설) 3 + 출처정리표 1. **검증 완료**(정답 균형·지문 무중복·서술 출제공식 반영).

## Context — 시험범위 & 비중

- 시험범위: **수능특강 Light 영어 9~14강** · **영어I 천재(조수경) 1~4과** · **고2 6월 학평 23~26년**.
- 26중간 유형 비중(객관식): 어휘5·어법4·빈칸4·요약3·의미추론(함의)2·순서2·삽입2·목적2·내용일치1·무관1·주제1. + 서술 4(우리말2·영작1·어법1).
- 파이널 변경점(사장님 지시): **1번** 목적→교과서3·4과 내용문제 / **4번** 목적→수특 빈칸 / **17·18** 모의 41~42 장문 **세트**(빈칸+어법) / 서술 출처 재배치.

## Details — 제작 규칙

- **출처 매칭**: 각 슬롯의 파이널 출처 카테고리 = 26중간 카테고리. 모의 N은 유형별 회전으로 분산(20번 쏠림 방지).
- **지문 무중복**: 차수 내 같은 지문 안 씀(생성 지문 포함 점유). 차수 간 생성 지문도 전역 유일.
- **정답 균형**: 자유형 15문항(제목·함의·빈칸·요약·주제·내용일치·순서)만 선지 순서 섞어 ①~⑤ 4~6개로 균형. 고정형(어법·어휘·무관·삽입=지문 내 위치)은 불변. **선지 순열 시 해설의 ①~⑤ 번호도 재매핑**.
- **서술형 출제공식 적용**([[서술형_출제원리_분석]]): 서술1=연결사 결과/이유절 추출 / 서술2=N가지 '~것'(부분점수) / 서술3=구동사 영작(파1 substitute A for B·파2 be confronted with 수동태·파3 deprive A of B) / 서술4=어법 5곳 중 수정+우리말 이유.
- **공식폼**: `formspec_동신여고2.json`(A4 2단·school_cover_abs 표지엔진) + `gen_exam.py`. 표지 유의사항 자동 보정(27문항80점·4문항20점·실제 쪽수).

## 담당 선생님 (출제자) — 차분 분석 키

- **25년 동신여고2 영어 = 천지영 · 윤영철**
- **26년 동신여고2 영어 = 천지영 · 정지원**
- → **천지영=25·26 공통(불변항)**. 누가 어떤 문항을 냈는지 라벨은 없으나, **25↔26 차분**으로 시그니처 분리 가능: 양쪽 일관=천지영 / 25만=윤영철 / 26신규=정지원. 26중간(천지영·정지원)이 우리 타깃.
- (※ '광주동신고'는 동신여고와 다른 학교 — 제외)
