# 동신여고3 SS.Verbs 단어장·단어시험 (자동 제작)

동신여고3 1학기 2차(영어Ⅱ) **SS.Verbs 시험범위 어휘**를 학습용 **단어장**과 **순서 섞은 단어시험(+정답본)** 으로 자동 제작. 파이널 시험지([[하위파이널_내신제작]])와 **별개 라인 = 어휘 학습/테스트 도구**. 같은 단어들은 [[하위보카]] 앱에도 반영. 방법(단일표 서식·다의어 뜻개수 빈칸·순서섞기)은 **다른 학교/범위 단어장에 그대로 재사용** 가능.

## Summary

- **완성 단어장 198어휘**: 기존 173 + `observe·parallel·order` 가족 + `dwell·draw·withdraw` 가족 25개를 **표제어 아래 행에 끼워넣고** 표제어 빈칸(★직접채우기) 5개(observe/parallel/order/dwell/draw)를 채움. withdraw는 (22번) 신설.
- **오타 5건** 멀티에이전트 교정(찾기→2인 독립검증, 둘 다 동의만 확정): `extensive`·`comprehensive`(옆 단어 뜻 복붙)·`serve`(역할을하다→역할을 하다)·`conserve`(아껴쓰다→아껴 쓰다)·`tenacious`(끈기있는→끈기 있는).
- **순서 섞은 단어시험 + 정답본**: 다의어는 뜻 개수만큼 `1. ___ 2. ___ …` 번호 빈칸, 단의어는 한 줄 빈칸. 시험지·정답 동일 순서.
- **[[하위보카]] 앱**에 동신여고3 20단어 추가(200단어) + 배포(sw v61, git push→Netlify 자동).

## Details — 제작 방법 (재사용 IP)

### 단어장 서식 (`render_dongsin2.py`)
- **단일 6열 표(2단, 가로로 읽기 1·2 / 3·4 …)** → 페이지 경계 자연 넘김. "50개씩 청크" 안 함(원장 지시).
- 열폭: **# 좁게 0.7cm** · 단어 2.7cm(Times New Roman 9pt 굵게 네이비 #132B4B) · 뜻 나머지(맑은 고딕 8pt).
- **네이비 헤더행**(`# 단어 뜻` ×2, 페이지마다 반복=`w:tblHeader`) · **표제어행 연파랑 음영**(#EAF1F8) · 연회색 격자(#D9D9D9) · 셀 세로중앙 · 셀여백 축소.
- 페이지 A4, 여백 L/R 1.1cm. 최종 4쪽.

### 다의어 뜻 개수 파싱 → 시험지 번호 빈칸
- `senses(뜻)` = "/"로 나눈 각 그룹의 `\d+\.` 마커 수(없으면 1)의 합.
  - 예: `order`("1.2.3.4 / (동)…")=5 · `produce`("생산하다 / (명) 농산물")=2 · `reduce`("줄이다, 감소시키다")=1.
- senses ≥ 2 → `1. ___ 2. ___ …` 번호 **밑줄 빈칸(문단 하단 테두리 pBdr)** · senses == 1 → 한 줄 빈칸.
- 다의어 32 · 단의어 166. 다의어 행은 **페이지 경계 분할 방지**(`w:cantSplit`).

### 순서 섞기 (`render_test.py`)
- `random.Random(시드=20260626)`로 셔플 → **시험지·정답본 동일 순서 보장**. **시드만 바꾸면 A/B형** 즉시 생성.
- 시험지는 표제어(N번)/[어근] 라벨·그룹 음영 제거(섞으면 무의미), # 는 문제번호로 1~198 순차.

### 오타 교정 (Workflow "dongsin-voc-proofread")
- 198항목 → **5청크 병렬 찾기** → 항목별 **2인 독립검증(둘 다 confirmed만 확정)**. 확정 5건만 반영, 보류 2건(reserve the food "남겨놓음"·unprecedented "유례 없는"=허용표기)은 손대지 않음.

### 렌더/검증
- python-docx 빌드. **시각 검증 = LibreOffice**(`soffice --headless --convert-to pdf`, ASCII 임시파일명으로 복사 후 변환=한글경로 인코딩 회피)→PyMuPDF PNG→읽기. (Word COM은 자동화서 hang → docx까지만 자동검증.)
