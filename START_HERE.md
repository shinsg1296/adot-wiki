## 일상 운영 (매 세션)

| 할 일 | 이렇게 말하면 됨 | 키워드 |
|---|---|---|
| 이전 맥락 불러오기 | "옵시디언 참조" | `query` |
| 이번 작업 저장하기 | "이번 작업 내용 옵시디언에 저장해줘" | `save` |
| 새 자료 정리하기 | "이 자료 ingest 해줘" | `ingest` |
| 위키 점검하기 | "위키 점검해줘" | `lint` |

- 세션은 `query`로 시작, `save`로 마무리하면 맥락이 이어집니다.
