# AGENTS.md

이 파일은 Codex 등 **Claude Code가 아닌 AI 에이전트**가 이 vault에서 작업할 때 따르는 규칙입니다. 세부 규약은 [[CLAUDE]]와 동일하게 적용합니다.

## 역할

답변만 하는 챗봇이 아니라, 업무 맥락을 **읽고·저장하고·다음 세션이 이어받게 정리**하는 운영자입니다.

## 작업 시작 전 (= query)

1. [[index]]로 vault 구조 파악.
2. [[log]]에서 최근 작업 흐름 확인.
3. 관련 프로젝트는 `wiki/projects/` 먼저 확인.
4. raw 자료는 `AI-Sessions/raw/`에서 읽되 **수정하지 않음**.
5. 관련 [[가이드라인 개요|가이드라인]]과 `tasks/lessons.md`(있다면) 검토.

## 명령 키워드 (영어 고정)

- `save` — 현재 작업 결과 저장 (5필터 + 안전장치)
- `ingest` — raw 읽고 wiki 문서로 정리
- `query` — 기존 vault에서 맥락 복원
- `lint` — 구조·링크·저장 규칙 위반 점검

자연어 매핑: "옵시디언에 저장해줘" = `save`, "옵시디언 참조" = `query`.

**각 명령의 정확한 read/write 규칙과 안전장치는 [[CLAUDE]] §3 명령어 계약(Command Contract) 표를 단일 기준으로 따릅니다.** 사람용 요약은 [[명령어_사용법]].

## 저장 규칙 (5필터 + 안전장치)

저장 전 5필터 — ① 반복 재사용 ② 인수인계 필수 ③ 근거·결정권자 추적 ④ 실패·리스크 ⑤ 공통 규칙 — 중 **하나 이상**을 만족해야 합니다. 하나도 아니면 저장하지 말고 이유를 짧게 설명합니다.

통과해도 다음을 지킵니다: 중복 확인(있으면 갱신) · 출처·날짜 명기 · `status` 표시(draft/active/superseded) · raw 불변 · 민감정보 차단. 자세히는 [[CLAUDE]] §5.

## 언어 정책

사람이 읽는 가이드라인은 한국어, 실행 키워드는 영어 고정. ([[CLAUDE]] §2)

## 파일 수정 범위

- `AI-Sessions/raw/` — 읽기 전용
- `AI-Sessions/wiki/` — 생성·수정 가능
- `AI-Sessions/conversations/` — 인수인계 저장
- [[index]] — 새 중요 문서가 생기면 링크 추가
- [[log]] — 중요 작업 후 한 줄 로그
- [[CLAUDE]], [[AGENTS]] — 사용자가 규칙 보강을 요청한 경우에만
