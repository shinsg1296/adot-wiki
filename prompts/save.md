# Save Prompt

자연어 트리거: "이번 작업 내용 옵시디언에 저장해줘" / "옵시디언에 저장"

```text
이번 작업 내용을 옵시디언에 저장(save)해줘.

저장 전에 CLAUDE.md §5의 저장 필터 5가지와 안전장치를 적용해줘.
- 5필터 중 하나라도 맞으면: 적절한 wiki 카테고리를 골라 문서를 만들고(같은 주제가 있으면 새로 만들지 말고 기존 문서 갱신), 출처·날짜·status·owner를 적고, index.md와 log.md도 갱신해줘.
- 하나도 안 맞으면: 저장하지 말고, 저장하지 않은 이유를 짧게 설명해줘. (필요하면 conversations에만 남겨줘)

raw 원본은 절대 수정하지 말고, 토큰·비밀번호·개인정보는 본문에 쓰지 말아줘.
```
