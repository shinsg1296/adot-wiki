# Lint Prompt

자연어 트리거: "위키 lint" / "위키 점검해줘"

```text
이 옵시디언 업무 위키를 lint 해줘. (자동 삭제하지 말고 제안 먼저)

CLAUDE.md §9 체크리스트로 점검해줘. 특히:
1. 검증 안 된 추측이 단정처럼 저장돼 있지 않은가
2. 중복 정보가 여러 문서에 흩어져 있지 않은가
3. 출처·결정권자 없는 결정이 있지 않은가
4. 새 자료로 대체된 낡은(stale) 규칙이 남아 있지 않은가

그 외: 문서 간 모순 · 고아 문서 · 누락된 상호 링크 · raw 경계 위반 · 민감정보 · index/log 누락.

문제는 파일별로 수정 제안을 정리해줘. 실제 수정은 내 승인 후에 해줘.
```
